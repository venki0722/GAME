# 🧠 Sliding Puzzle Game

This is a brain game built using Python and Streamlit.

## 🎮 Game Description

* 4x4 sliding puzzle
* Arrange numbers from 1 to 15
* One empty space to move tiles

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## 📌 Features

* Move counter
* Restart button
